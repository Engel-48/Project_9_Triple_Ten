# Proyecto Urban Grocers
Buenas, en este proyecto se ejecutan 2 clases. La primera llamada UrbanRoutesPage
ejecuta 19 métodos y la última parte de esta clase dónde se ejecutan métodos para
comprobar assert. 
Y en la segunda clase TestUrbanRoutes, ejecutan las 20 pruebas necesarias para configurar 
el requiriiento de un taxi en la aplicación de Urban.Routes.

Tiene como peculiaridad de finalizar el proyecto con un @classmethod.

Se usaron los paquetes de Selenium, Webdriver, WebDriver.Wait
Se usaron los archivos data.py
